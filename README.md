## INSTALACION

### :one:. Descargar y correr en development

```bash
git clone https://github.com/jotalevi/oceanic.git
cd oceanic
flutter pub get
flutter run
```

### :two:. Build

#### Android
```bash
flutter build apk
```

#### iOS
```bash
flutter build ipa
```
